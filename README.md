# 💰 MyMoney Tracker

MyMoney Tracker is a personal finance dashboard web application that helps users track their income, expenses, and overall financial activity in one place.

The platform allows users to add transactions, monitor their balance, and analyze their spending patterns through a clean and modern dashboard interface.

This project was built as a **portfolio project using HTML, CSS, and JavaScript**.

## ✨ Features

- Track **total income, expenses, and net balance**
- Add and manage income or expense transactions
- View **recent transaction history**
- Categorize expenses for better financial tracking
- Dashboard cards showing financial overview
- Spending analytics through charts and visual summaries
- Clean **dark-themed modern dashboard UI**
- Responsive layout that works on desktop and mobile devices

## 🛠 Tech Stack

- **HTML5** – structure of the website  
- **CSS3** – styling and dashboard layout  
- **Vanilla JavaScript** – transaction handling and dynamic updates  
- **LocalStorage API** – storing transaction data in the browser

## 📖 How It Works

1. The dashboard displays the **current financial overview**, including net balance, total income, total expenses, and savings rate.
2. Users can click **Add Transaction** to enter new financial records.
3. Each transaction includes:
   - amount
   - type (income or expense)
   - category
   - description
   - date
4. Transactions are stored in the browser using **localStorage**.
5. The dashboard automatically updates the **financial summary cards**.
6. Recent transactions appear in the **transaction history section**.
7. Spending data is visualized in the **analytics section**.
